# marathon-theme-zed

A theme for the Zed text editor based on the art desing of the 2026 video game Marathon by Bungie.

## Installation

To install this theme, search for it in the Zed extension store and click "Install." Then, set the theme in the settings to `Marathon Dark` or `Marathon Light`.

### Manual install

To use this theme manually, copy the contents of `themes/marathon.json` into your `~/.config/zed/themes` folder.

See https://zed.dev/docs/themes?highlight=themes#themes for more info.
